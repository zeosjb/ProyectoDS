# Arquitectura

- Next.js App Router con Server Components por defecto.
- Supabase SSR mediante cookies y `@supabase/ssr`.
- Mutaciones sensibles en Server Actions o funciones PostgreSQL.
- Builder.io edita landing, secciones publicas y componentes registrados.
- Cada carpeta tiene dependencias propias y puede desplegarse por separado en Vercel.
