# Pruebas

Ejecutar:

```bash
npm install
npm run lint
npm run typecheck
npm run build
npm test
```

Validaciones manuales:

- `.env.local` esta ignorado.
- Los formularios muestran mensajes claros en espanol.
- Las tablas publicas tienen RLS habilitado.
- Builder compila sin API Key y muestra respaldo local.
