# Tablero Prisma

Plantilla Kanban academica con integrantes, responsables, prioridades y vista de tabla.

## Inicio rapido

1. Crea un repositorio desde esta carpeta template.
2. Usa la version de Node.js disponible en tu equipo o deployment y ejecuta `npm install`.
3. Crea un proyecto Supabase independiente.
4. Copia `.env.example` como `.env.local`.
5. Copia la URL y la Publishable Key de Supabase.
6. Ejecuta `npm run supabase:link`.
7. Ejecuta `npm run supabase:push`.
8. Ejecuta `npm run supabase:types`.
9. Crea un espacio Builder.io independiente.
10. Configura `NEXT_PUBLIC_BUILDER_API_KEY`.
11. Ejecuta `npm run dev`.
12. Despliega en Vercel.
13. Configura las variables en Vercel.
14. En Supabase Auth configura Site URL y Redirect URLs con tu dominio local y productivo.
